Module: teamsRoutes

Router for all requests to '/teams'. GET: Accepts the follow queries: 'id' - Return team by Id 'id & games' - Return games involving the team with Id 'lid' - Return all teams in a given league 'lid & stats' - Return all teams in a given league with stats 'tn' - Return teams by teamname '' - Returns all users POST: Takes a teamname and leagueId and creates a new team. PUT: Takes two team Ids, two scores and creates a game DELETE: Accepts the follow queries: 'id' - Deletes a team
